# Active Owner

## Purpose
This file is the coordination point for live server ownership on the shared host.

Rule:
Only one active live operator may hold server ownership at a time.

## Status Template
- Live owner: `unclaimed`
- Project/repo: `ai-assistant`
- Mode: `off-server`
- Started at UTC: `not set`
- Expected handoff at UTC: `not set`
- Task or branch: `not set`
- Notes: `claim this file before opening a heavy live remote session`

## Usage
- Before starting live work, update this file in your branch or task context.
- If you do not need live server access, leave mode as `off-server`.
- On handoff, record the next owner or mark the slot as `unclaimed`.
- If ownership is unclear, do not start another live Remote SSH AI session.
